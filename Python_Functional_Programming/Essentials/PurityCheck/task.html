You are given with an increment function which accepts Integer class and return version of Integer with its previous value + 1.

For example:

increment(Integer(1)) should return Integer(2)

increment(Integer(2)) should return Integer(3)

increment(increment(Integer(1))) should return Integer(3)

The following should also be true 
  
    val input = Integer(1)
    val res1 = increment(input)
    val res2 = increment(input)

    res1 == res2
Task:

Write a test that check if the given transformation is Pure.
